# 🛡️ DDoS Block Dark - Advanced Protection
Created by: **Dara Dav (imDara)**

### 🚀 Features
- **Real-time Monitoring:** Tracking requests/sec and requests/min.
- **Glassmorphism UI:** Modern Dashboard built with Flask.
- **Auto-Firewall:** Integrates with Windows Firewall & Iptables.
- **Data Persistence:** Saves blocked IPs to JSON.

### 🛠️ Installation
1. Clone the repo: `git clone https://github.com/yourusername/ddos-block-dark.git`
2. Install dependencies: `pip install flask`
3. Run: `python ddos_dark.py`
